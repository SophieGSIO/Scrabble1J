# Scrabble1J
Projet SCRABBLE - Mode 1 joueur 

## Description
- **Jeu simplifié du SCRABBLE** : l'application affiche le score du mot saisi par l'utilisateur / [Valeur des lettres au Scrabble](https://fr.wikipedia.org/wiki/Lettres_du_Scrabble "Wikipédia")
- Application de bureau **WPF / C#**
- Traitement des **demandes de corrections et d'évolutions**
